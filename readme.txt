Plug-in to create simple to post demo page that is already published.

== Description ==

This is a plug-in to create simple to post demo page that is already published. This edit screen of Posts and fixed page as a plug-in to enable you to display the editor for editing demo page. By to enter the demo content to demo page editing editor update, you will be able to easily create a demo page. After that I just click the demo view button.

== Installation ==

1. Upload the `wp-post-demo` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the Plugins menu in WordPress.
